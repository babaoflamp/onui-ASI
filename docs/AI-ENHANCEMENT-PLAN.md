# 🚀 오누이 한국어 AI 서비스 강화 로드맵 (AI-ENHANCEMENT-PLAN)

이 문서는 오누이 한국어 플랫폼의 AI 기능을 강화하여 사용자 경험(UX)을 혁신하고 학습 효과를 극대화하기 위한 상세 구현 계획을 담고 있습니다.

---

## 📅 프로젝트 개요
*   **목표:** 단순 학습 플랫폼을 넘어 '지능형 AI 튜터'로 진화
*   **핵심 가치:** 실전 몰입, 시각적 피드백, 개인 맞춤형 학습, 일상 밀착형 확장

---

## 🛠 4대 강화 전략 및 상세 계획

### 1. 실전 몰입형: AI 상황별 롤플레잉 (AI Roleplay)
사용자가 한국어로 특정 상황에서 AI와 자유롭게 대화하며 실전 감각을 익히는 기능입니다.

*   **기능 상세:**
    *   상황 설정: 카페, 공항, 면접, 첫 만남 등 시나리오 선택
    *   대화 엔진: Gemini 2.0 Flash 또는 Ollama (Exaone) 활용 (Context 유지)
    *   실시간 평가: 발음(SpeechPro) + 상황 적절성(LLM) 동시 채점
    *   대화 종료 후 피드백 리포트 제공 (더 나은 표현 추천)
*   **기술 스택:**
    *   `backend/routes/roleplay.py`: 대화 세션 관리 API
    *   `templates/ai-roleplay.html`: 채팅형 UI 및 음성 인식 연동
    *   `LLM Prompt Engineering`: 한국어 튜터 역할 부여

### 2. 초개인화: AI 약점 진단 및 오답 지도 (AI Weakness Diagnosis)
사용자의 과거 학습 데이터를 분석하여 개인별 맞춤형 복습 경로를 제시합니다.

*   **기능 상세:**
    *   취약 발음 분석: 자주 틀리는 음소(예: 'ㄹ', 'ㅢ') 자동 추출 및 통계화
    *   지능형 복습 시스템: 에빙하우스 망각 곡선 기반 취약 단어 재노출
    *   개인별 AI 요약 리포트: 주간 학습 성취도 및 집중 학습 영역 추천
*   **기술 스택:**
    *   `SQLite (word_score_history, sentence_score_history)` 데이터 마이닝
    *   `backend/services/analytics_service.py`: 개인별 취약점 분석 로직
    *   `templates/my-weakness.html`: 데이터 시각화 대시보드 (Chart.js 활용)

### 3. 확장성: 시각 지능 기반 학습 (Visual-to-Korean)
카메라로 사물을 찍으면 AI가 한국어 단어와 예문을 생성해주는 서비스입니다.

*   **기능 상세:**
    *   사물 인식: 사용자가 업로드한 이미지에서 객체 추출
    *   콘텐츠 생성: 사물 이름(한국어/영어/발음) + 해당 사물을 포함한 상황별 예문 3개 생성
    *   학습 연동: 생성된 단어를 즉시 단어장에 추가 가능
*   **기술 스택:**
    *   `Gemini 2.0 Flash (Multimodal)` API 활용 (Image input 지원)
    *   `backend/routes/vision_learning.py`: 이미지 처리 및 LLM 연동 API
    *   `templates/vision-capture.html`: 모바일 카메라 연동 및 인식 UI

### 4. 시각적 혁신: AI 디지털 휴먼 튜터 (Lightweight Lip-sync)
정적인 캐릭터를 넘어 생동감 있게 말하고 반응하는 AI 아바타를 도입합니다.

*   **기능 상세:**
    *   입모양 동기화 (Lip-sync): TTS 출력에 맞춰 캐릭터의 입 모양 애니메이션 구현
    *   감정 표현: 정답/오답 상황에 따른 캐릭터 표정 변화 (SVG/Sprite 애니메이션)
    *   가상 시연: 어려운 발음 시 입 모양을 확대하여 정확한 조음 위치 가이드
*   **기술 스택:**
    *   `Phoneme-to-Viseme Mapping`: 한국어 음소와 입 모양 매핑 로직
    *   `CSS Animation & SVG`: 가벼운 웹 기반 애니메이션 처리
    *   `HeyGen API` (선택 사항): 고화질 영상 필요 시 연동 검토

## 📈 단계별 구현 일정

| Phase | 단계 | 주요 작업 내용 | 예상 소요 시간 |
| :--- | :--- | :--- | :--- |
| **Phase 1** | **AI 롤플레잉** | 백엔드 대화 로직, 상황별 프롬프트, UI 구현 | 3일 |
| **Phase 2** | **약점 진단** | DB 쿼리 최적화, 분석 알고리즘, 대시보드 구현 | 2일 |
| **Phase 3** | **시각 학습** | Gemini Multimodal 연동, 이미지 업로드/인식 UI | 2일 |
| **Phase 4** | **디지털 휴먼** | 음소-입모양 매핑, SVG 애니메이션 고도화 | 3일 |

## 🎯 기대 효과
1.  **사용자 유지율(Retention) 향상:** 개인 맞춤형 피드백을 통한 성취감 제공
2.  **기술적 차별화:** 단순 스코어링을 넘어선 상황 판단 및 시각 지능 결합
3.  **학습 효율 극대화:** 실전 상황 노출을 통한 빠른 발화 능력 향상
